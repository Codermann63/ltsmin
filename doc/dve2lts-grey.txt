== NAME

dve2lts-grey - Sequential state space generator for DIVINE models.

== SYNOPSIS

*dve2lts-grey* [OPTION]... 'input'[_.dve_|_.dveC_] ['output.fmt']

== DESCRIPTION

dve2lts-grey generates a labelled transition system from a
specification provided in 'input.dve/input.dveC' by enumerative reachability
analysis. 'output.fmt' specifies the name of the output archive.  The desired
format is deduced from the filename extension.
