*-d*, *--deadlock*::
      Find state with no outgoing transitions.  Returns with exit code
      1 if deadlock is found, otherwise 0.

*--trace*='FILE'_.gcf_::
      When finding a deadlock state, write a trace to 'FILE'_.gcf_,
      beginning at the initial state.   Traces can be pretty-printed
      with linkman:ltsmin-tracepp[1].
