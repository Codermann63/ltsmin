== NAME

dve2lts-mpi - Distributed state space generator.

== SYNOPSIS

mpirun _<nodespec>_ *dve2lts-mpi* [OPTION]... 'input'[_.dve_|_.dveC_] ['output.fmt']

== DESCRIPTION

dve2lts-mpi performs a reachability analysis on a specification provided
in 'input.dve/input.dveC'.  Specifications are in _dve_ format, or in _dveC_
format generated by the divine.precompile tool.  'output.fmt' specifies the name of
the output archive.  The desired format is deduced from the filename
extension.
