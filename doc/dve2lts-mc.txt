== NAME

dve2lts-mc - Concurrent state space generator for DiVinE (cluster) models.

== SYNOPSIS

*dve2lts-mc* [OPTION]... 'input'[_.dve_|_.dveC_] ['output']

== DESCRIPTION

dve2lts-mc generates a labelled transition system from a
specification provided in 'input.dve' or 'intput.dveC' (precompiled).
It does this concurrently for any given number of threads.
