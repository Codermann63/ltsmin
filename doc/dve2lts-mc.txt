== NAME

nips2lts-mc - Concurrent state space generator for DIVINE (cluster) models.

== SYNOPSIS

*dve2lts-mc* [OPTION]... 'input.dve' ['output']

== DESCRIPTION

dve2lts-mc generates a labelled transition system from a
specification provided in 'input.dve'. It does this concurrently
any given number of threads.
