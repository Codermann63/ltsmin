== NAME

spinjal - compile promela models to .spinja PINS libraries for use with
prom frontends.

== SYNOPSIS

*{manname}* [-o3] model.promela

*{manname}* [-o3] model.prm

*{manname}* [-o3] model

*{manname}* [-o3] model.prom

*{manname}* [-o3] model.pml

== DESCRIPTION

{manname} executes the SpinJa compiler, that is part of the LTSmin distribution,
on the given Promela model. The result is an .spinja object file that implements
the PINS interface and can be used by the +prom+ frontends.


== OPTIONS
*-v*::
        Increase verbosity.
