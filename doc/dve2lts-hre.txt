== NAME

dve2lts-hre - Distributed state space generator for DIVINE models.

== SYNOPSIS

*dve2lts-hre* [OPTIONS]... 'input'[_.dve_|_.dveC_] ['output.fmt']

== DESCRIPTION

dve2lts-hre performs a reachability analysis on a specification provided
in 'input.dve/input.dveC'.  Specifications are in _dve_ format, or in _dveC_
format generated by the divine.precompile tool.  'output.fmt' specifies the
name of the output archive.  The desired format is deduced from the filename
extension.
