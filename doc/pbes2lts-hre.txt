== NAME

pbes2lts-hre - Distributed parity game generator.

== SYNOPSIS

*{manname}* [OPTIONS]... 'input.pbes' ['output.fmt']

== DESCRIPTION

{manname} performs a reachability analysis on a specification provided
in 'input.pbes'.  Specifications are in _pbes_ format and are commonly
generated by manpage:lps2pbes[1].  'output.fmt' specifies the name of
the output archive.  The desired format is deduced from the filename
extension. Available formats are described below.
