== NAME

lps2lts-gsea - Sequential state space generator for mCRL2 models.

== SYNOPSIS

*{manname}* [OPTION]... 'input'_.lps_

== DESCRIPTION

{manname} generates a labelled transition system from a specification
provided in 'input.lps' by enumerative reachability analysis using a
general state expanding algorithm. Specifications are in _lps_ format
and are commonly generated by manpage:mcrl22lps[1].
