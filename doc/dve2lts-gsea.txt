== NAME

dve2lts-gsea - Sequential state space generator for DIVINE models.

== SYNOPSIS

*{manname}* [OPTION]... 'input'[_.dve_|_.dve2C_]

== DESCRIPTION

{manname} generates a labelled transition system from a specification
provided in 'input.dve' or 'input.dve2C' by enumerative reachability
analysis using a general state expanding algorithm.
