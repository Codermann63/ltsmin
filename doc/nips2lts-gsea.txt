== NAME

nips2lts-gsea - Sequential state space generator for NIPS models.

== SYNOPSIS

*{manname}* [OPTIONS]... 'input'_.b_

== DESCRIPTION

{manname} generates a labelled transition system from a specification
provided in 'input.b' by enumerative reachability analysis using a
general state expanding algorithm.
