== NAME

dve22lts-grey - Sequential state space generator for DiVinE DVE models.

== SYNOPSIS

*{manname}* [OPTION]... 'input'[_.dve_|_.dve2C_] ['output.fmt']

== DESCRIPTION

{manname} generates a labelled transition system from a specification
provided in 'input.dve' or 'input.dve2C' by enumerative reachability
analysis.  'output.fmt' specifies the name of the output archive.  The
desired format is deduced from the filename extension.
