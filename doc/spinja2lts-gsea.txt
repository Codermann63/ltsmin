== NAME

spinja2lts-gsea - Sequential state space generator for SpinJa models.

== SYNOPSIS

*{manname}* [OPTION]... 'input.spinja'

== DESCRIPTION

{manname} generates a labelled transition system from a specification
provided in 'input.spinja' by enumerative reachability
analysis using a general state expanding algorithm.
