== NAME

ltscmp-one - comparison of labelled transition systems modulo bisimulation

== SYNOPSIS

ltscmp-one [OPTIONS] <input 1> <input 2>

== OPTIONS

*-s*::
Compare modulo strong bisimulation using the default implementation.

*--strong[='variant']*::
Compare modulo strong bisimulation using the given 'variant' or the default implementation.

*-b*::
Compare modulo branching bisimulation using the default implementation.

*--branching[='variant']*::
Compare modulo branching bisimulation using the given 'variant' or the default implementation.

*-l*, *--lump*::
Compare modulo CTMC lumping.

include::inc/hre-options.txt[]

== EXIT STATUS
0::
        Successful termination: LTSs are equivalent.
>0::
        Some error occurred or LTSs are not equivalent.
