== NAME

dve-reach - Symbolic reachability analysis for DIVINE models.

== SYNOPSIS

*dve-reach* [OPTION]... 'input'[_.dve_|_.dveC_] ['output.etf']

== DESCRIPTION

dve-reach performs a reachability analysis on a specification provided
in 'input.dve/input.dveC'.  Specifications are in _dve_ format, or in
_dveC_ format generated by the divine.precompile tool. 'output.etf' specifies the
name of the output archive, written in linkman:etf[5] format.
