== NAME

etf2lts-mc - Concurrent state space generator for ETF models.

== SYNOPSIS

*etf2lts-mc* [OPTION]... 'input.etf' ['output']

== DESCRIPTION

etf2lts-mc generates a labelled transition system from a compiled
specification provided in 'input.etf'. It does this concurrently
for any given number of threads.
