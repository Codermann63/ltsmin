== NAME

spinja2torx - TorX RPC interface for SPINJA models.

== SYNOPSIS

*{manname}* [OPTION]... 'input.spinja'

== DESCRIPTION

{manname} provides access to a labelled transition system from a
specification provided in 'input.spinja' via the TorX-Explorer textual
interface on _stdin_ and _stdout_.  Specifications are compiled with
SpinJa from PROMELA input models.
