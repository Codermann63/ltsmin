== NAME

prom2lts-mc - Concurrent state space generator for SpinJa models.

== SYNOPSIS

*{manname}* [OPTION]... 'input.prom'

== DESCRIPTION

{manname} generates a labelled transition system from a compiled
specification provided in 'input.prom'.  It does this concurrently
for any given number of threads.
