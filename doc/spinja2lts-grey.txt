== NAME

spinja2lts-grey - Sequential state space generator for SpinJa models.

== SYNOPSIS

*{manname}* [OPTION]... 'input.spinja' ['output.fmt']

== DESCRIPTION

{manname} generates a labelled transition system from a specification
provided in 'input.spinja' by enumerative reachability analysis.
Specifications are compiled with SpinJa from PROMELA input models.
'output.fmt' specifies the name of the output archive.  The desired
format is deduced from the filename extension.
