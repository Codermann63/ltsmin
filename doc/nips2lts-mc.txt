== NAME

nips2lts-mc - Concurrent state space generator

== SYNOPSIS

*nips2lts-mc* [OPTION]... 'input.b'

== DESCRIPTION

nips2lts-mc generates a labelled transition system from a
specification provided in 'input.b'. It does this concurrently
for any given number of threads.
