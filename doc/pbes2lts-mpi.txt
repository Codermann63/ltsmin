== NAME

pbes2lts-mpi - Distributed parity game generator

== SYNOPSIS

mpirun _<nodespec>_ *{manname}* [OPTION]... 'input.pbes' ['output.fmt']

== DESCRIPTION

{manname} generates a parity game from a
specification provided in 'input.pbes'.  
Specifications are in _pbes_ format and are commonly
generated by manpage:lps2pbes[1].  'output.fmt' specifies the name
of the output archive.  The desired format is deduced from the
filename extension.
