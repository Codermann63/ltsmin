== NAME

nips2torx - TorX RPC interface for NIPS models.

== SYNOPSIS

*{manname}* [OPTION]... 'input.b'

== DESCRIPTION

{manname} provides access to a labelled transition system from a
specification provided in 'input.b' via the TorX-Explorer textual
interface on _stdin_ and _stdout_.
