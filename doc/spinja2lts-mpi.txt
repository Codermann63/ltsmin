== NAME

spinja2lts-mpi - Distributed state space generator for SPINJA models.

== SYNOPSIS

mpirun _<nodespec>_ *{manname}* [OPTION]... 'input.spinja' ['output.fmt']

== DESCRIPTION

{manname} generates a labelled transition system from a specification
provided in 'input.spinja'.  Specifications are compiled with SpinJa
from PROMELA input models.  'output.fmt' specifies the name of the
output archive.  The desired format is deduced from the filename
extension.
