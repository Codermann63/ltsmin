== NAME

pbes-reach - Symbolic reachability analysis for PBES models.

== SYNOPSIS

*{manname}* [OPTION]... 'input.pbes' ['output.etf']

== DESCRIPTION

{manname} performs a reachability analysis on a specification provided
in 'input.pbes'.  Specifications are in _pbes_ format and are commonly
generated by manpage:lps2pbes[1].  'output.etf' specifies the name
of the output archive, written in manpage:etf[5] format.
